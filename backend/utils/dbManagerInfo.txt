CreateNewDB:
--> Creates a new table for the new data source
--> if data source tech stack already exists: add table to existing tech stack database

startUpDataValidation:
--> checks orderedData vs rawData for discrepancies 
--> if discrepancy, return notification.
--> returns 1 of 3 cases:
--> --> case 1: all data sources is inside ordered data. returns true
--> --> case 2: tech stack item database doesn't exist, returns: [tech stack item name, table name]

Variable formats:
--> tData:
        [
            [row1Attr1,row1Attr2,etc],
            [row2Attr1,row2Attr2,etc],
            [row3Attr1,row3Attr2,etc]
        ]
--> tName (table name): lowercase String
--> tHeaders: [header1,header2,header3,etc]
--> tHeadersType: [header1datatype, header2datatype, et]
--> techStackItemName (Tech stack item name): lowercase string 
--> tType (CRM or analytics): lowercase string 
--> attributes: [attribute1, attribute2, etc]
--> conditions: [conditionForAttribute1, conditionForAttribute2, etc]

getSpecificData:
--> Attributes: ["attribute1", "attribute2", etc]
--> conditions: ["(result)", false, etc]
--> --> Using the example above, attribute1 must equal (result), attribute2 doesn't have a condition.
